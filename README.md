# Console File Manager (C++)

A simple console-based file manager written in **C++** using the std::filesystem library. This project was built to practice object-oriented programming, file I/O, and filesystem operations.

## Features

*  Show the current working directory
*  List files and directories
*  Create new files
*  Optionally write text into newly created .txt files
*  Delete files
*  Rename files
*  Create new directories
*  Change the current directory
*  Go back to the parent directory
*  Input validation for the menu
